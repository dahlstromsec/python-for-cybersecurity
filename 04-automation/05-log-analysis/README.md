# 05 Log Analysis

This section focuses on using Python to collect, parse and analyze log files.

Log analysis is one of the most valuable skills in cybersecurity. Security analysts regularly investigate authentication logs, web server logs, firewall logs and system logs to identify suspicious activity, troubleshoot issues and support incident response.

- Filtering events
- Extracting IP addresses
- Searching for keywords
- Failed login detection
- Log statistics
- Regular Expressions (Regex)
- IOC (Indicator of Compromise) detection

Develop practical Python skills for analyzing logs and extracting meaningful information from large datasets commonly encountered in cybersecurity.
